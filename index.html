Abstract

Redefining How Mobile Devices
Choose Their Network

The rapid growth of mobile data consumption has intensified performance challenges in LTE networks, particularly where multiple frequency bands are deployed. In Sri Lanka, mobile devices automatically select LTE bands based purely on signal strength — without considering real-time congestion or temporal performance patterns.

This results in unbalanced load distribution across available bands, degrading user experience despite adequate network coverage. Our research proposes an Intelligent LTE Band Recommendation System powered by crowdsourced spatio-temporal performance analysis to address this critical gap.

By leveraging real-world LTE data — signal metrics, throughput, location, and time-of-day parameters — combined with a Qwen 3 32B LLM with RAG architecture, we provide context-aware, actionable band recommendations that significantly improve individual user throughput.

Research Problem

The Core Challenge

Despite LTE networks covering multiple frequency bands in Sri Lanka, users consistently experience performance bottlenecks that trace back to a single root cause.

📶

Signal ≠ Speed

Current LTE band selection relies solely on RSRP (signal strength), directing users to the strongest signal without accounting for band congestion or load.

⚡

Peak-Hour Collapse

During lunch and evening peaks, automatic selection fails catastrophically — causing speeds to drop over 3x while better bands sit underutilized nearby.

🗺️

No Location Context

Band performance varies dramatically across geography and time. Existing tools offer no spatio-temporal intelligence — only raw metrics that non-technical users can't act on.

🇱🇰

Sri Lanka Data Gap

No systematic study exists for LTE band performance in Sri Lanka, despite unique multi-operator deployment patterns by Dialog, Mobitel, and Hutch.

App Features

What the App Does

🤖

AI Band Recommendation

Qwen 3 32B LLM with RAG architecture analyzes your location, time slot, and crowdsourced historical data to recommend the optimal LTE band with a human-readable explanation.

Groq Cloud RAG Confidence Score
📡

Real-Time Signal Telemetry

Continuously monitors RSRP, RSRQ, SINR, Band ID, eNodeB Tower ID, and Carrier Aggregation data from your device in real time.

⚡

Professional Speed Testing

Dual-mode testing: Auto HTTP benchmarking or manual entry of Ookla results. Measures download, upload, and latency with an animated speed gauge.

🗺️

Interactive Tower Map

Visualizes 5,000+ LTE towers across Sri Lanka using OpenStreetMap. Shows Dialog (2,200+), Mobitel (1,400+), and Hutch (1,200+) infrastructure.

📊

ISP Speed Leaderboard

Real-time crowdsourced ranking of ISP performance by location and time slot, showing which operator delivers the best speeds near you right now.

☁️

Crowdsourced Intelligence

Every user's speed test contributes to the shared knowledge base. The more users, the smarter the recommendations become for everyone.
